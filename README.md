# Punto-De-Venta_PF
El sistema desarrollado es una aplicación de Punto de Venta orientada a la gestión y venta de productos tecnológicos dentro del negocio ficticio llamado Chapi’s.
 Fue implementado en Java utilizando el paradigma de Programación Orientada a Objetos (POO) y una interfaz gráfica basada en Swing.
El objetivo principal del sistema es permitir que dos tipos de usuarios, Administrador y Cliente, interactúen con las distintas funciones del punto de venta a través de una interfaz amigable y módulos bien organizados.
El Administrador es responsable del manejo de los productos dentro del inventario del negocio. Entre sus funcionalidades destacan:
Iniciar sesión como administrador mediante credenciales predefinidas.


Registrar nuevos productos proporcionando ID, nombre, precio y stock.


Actualizar stock cuando intenta dar de alta un producto existente.


Consultar el inventario visualizando una tabla con todos los productos almacenados.


El cliente puede navegar y realizar compras dentro del sistema mediante las siguientes acciones:
Registrarse o continuar como invitado.


Visualizar el catálogo de productos disponible en el inventario.


Agregar productos a su carrito de compras, siempre y cuando exista stock disponible.


Modificar el carrito, ya sea quitando productos o vaciándolo por completo.

La aplicación cuenta con diversas ventanas creadas mediante Java Swing, organizadas modularmente:
Menú principal; Menú administrador; Formulario de agregar productos; Visualización del inventario; Menú del cliente; Carrito de compras; Pantalla de pago.
